# Introduction

opsbench is an open-source agent toolkit for operations teams.

Each "team" is a self-contained package that ships a coherent set of skills, subagents, schemas, Cedar policies, hooks, and MCP recipes. Together they teach a coding agent (Claude Code or Codex CLI) to behave like a senior engineer in that discipline — not by guessing, but by following a documented workflow with explicit gates, least-privilege tool scoping, and schema-validated outputs.
