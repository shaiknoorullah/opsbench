# F3 — Signed receipts (evidence v2) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax. Each step is sized to 2-5 min of agent work; the largest are the canonical-JSON fixture-corpus build in Task 4 and the recipe-snippet backfill in Task 11. Treat all commands as executed from the repo root unless a step explicitly `cd`s.

**Goal:** Ship Ed25519-signed custody receipts (schema v2) across both the F2 `opsbench-gateway` writer and the legacy `post-tool-use.sh` hook path, together with a key-bootstrap stage in `scripts/install.sh`, a dual-surface offline verifier (Bash + gateway subcommand) sharing exit codes and a fixture corpus, optional offsite mirroring with WORM enforcement, a key-rotation runbook, and CI that proves tamper-detection on every PR — without renaming, narrowing, or removing any F2 v1 custody-log field.

**Architecture:** Three writer paths converge on one envelope (`receipt.v1.json` inlined as `custody-log.v2.json`): (1) the gateway's custody writer calls a new `internal/signing/` package after the F2 v1 fields are assembled, computing a canonical-JSON SHA-256 then `ed25519.Sign`; (2) the legacy hook sources `lib/sign-receipt.sh` which shells `openssl pkeyutl -sign` over the same canonical payload and appends to a sibling `receipts.jsonl`; (3) the optional S3/Azure-Blob/GCS mirror replays the same lines append-only with bucket-level object-lock required at startup. The verifier rebuilds the canonical bytes, recomputes SHA-256, and runs Ed25519 verify, walking `parent_receipt_sha256` to flag broken chains. Key bootstrap is idempotent and lives in `scripts/install.sh`; rotation is a graceful restart documented in `key-rotation-runbook.md`.

**Tech Stack:** Go 1.22+ (gateway, `crypto/ed25519`, `encoding/json` with deterministic key sort), Bash 4+ (hook + verifier + installer), `openssl` (Ed25519 sign/verify on the Bash path), `jq` (canonical JSON in Bash via `-cS`), `xxd` (hex encoding), `uuidgen` or `python3 -c 'import uuid'` (UUIDv4 fallback), `aws-cli` or `azcopy` or `gcloud storage` (mirror tests via LocalStack/Azurite/fake-gcs-server in CI), `ajv-cli` (schema validation in CI), GitHub Actions, lefthook (existing).

  ```bash
  ensure_signing_keys() {
      local keys_dir="${HOME}/.config/opsbench/keys"
      local priv="${keys_dir}/gateway.key"
      local pub="${keys_dir}/gateway.pub"
      local sid="${keys_dir}/gateway.id"

      mkdir -p "${keys_dir}"
      chmod 0700 "${keys_dir}"

      if [[ -f "${priv}" && -f "${pub}" && -f "${sid}" ]]; then
          ok "signing keys present at ${keys_dir}"
          local existing_fp
          existing_fp="$(openssl pkey -in "${priv}" -pubout -outform DER \
                | openssl dgst -sha256 -binary | xxd -p -c 256)"
          ok "existing pubkey fingerprint=sha256:${existing_fp}"
          return 0
      fi

      info "generating Ed25519 signing key at ${priv}"
      require_cmd openssl
      if [[ "${DRY_RUN:-0}" == "1" ]]; then
          info "[dry-run] would write ${priv} (mode 0600), ${pub} (0644), ${sid} (0644)"
          return 0
      fi
      openssl genpkey -algorithm Ed25519 -out "${priv}" 2>/dev/null
      chmod 0600 "${priv}"
      openssl pkey -in "${priv}" -pubout -out "${pub}"
      chmod 0644 "${pub}"

      if command -v uuidgen >/dev/null 2>&1; then
          uuidgen | tr 'A-Z' 'a-z' > "${sid}"
      else
          python3 -c 'import uuid; print(uuid.uuid4())' > "${sid}"
      fi
      chmod 0644 "${sid}"

      local fp
      fp="$(openssl pkey -in "${priv}" -pubout -outform DER \
            | openssl dgst -sha256 -binary | xxd -p -c 256)"
      ok "signer_id=$(cat "${sid}")"
      ok "pubkey fingerprint=sha256:${fp}"
      info "share the fingerprint with auditors out-of-band; the private key never leaves this host"
  }
  ```

## Task 3 — PR 1: Bash verifier `scripts/verify-receipts.sh`

**Files:** Create — `scripts/verify-receipts.sh`, `scripts/verify-receipts.test.sh`.

- [ ] **3.1 — Write `scripts/verify-receipts.sh`** (mode 0755). It implements the algorithm from spec § 3.7. Pseudocode → real script:

  ```bash
  #!/usr/bin/env bash
  set -euo pipefail

  PUBKEY="${HOME}/.config/opsbench/keys/gateway.pub"
  PUBKEY_MAP=""
  STRICT=0
  INPUT=""

  die() { echo "verify-receipts: $*" >&2; exit "${2:-3}"; }

  while [[ $# -gt 0 ]]; do
    case "$1" in
      --pubkey)     PUBKEY="$2"; shift 2 ;;
      --pubkey-map) PUBKEY_MAP="$2"; shift 2 ;;
      --strict)     STRICT=1; shift ;;
      -h|--help)
        sed -n '1,40p' "$0" >&2; exit 0 ;;
      *)            INPUT="$1"; shift ;;
    esac
  done

  [[ -n "$INPUT" && -r "$INPUT" ]] || die "input log not readable" 3
  [[ -r "$PUBKEY" ]] || die "pubkey not readable at $PUBKEY" 3

  command -v jq      >/dev/null || die "jq required" 3
  command -v openssl >/dev/null || die "openssl required" 3
  command -v xxd     >/dev/null || die "xxd required" 3

  total=0; signed=0; unsigned=0; failed=0
  declare -A SEEN_HASHES   # for parent-chain walking within a single file

  tmpdir="$(mktemp -d)"
  trap 'rm -rf "$tmpdir"' EXIT

  lineno=0
  while IFS= read -r line; do
    lineno=$((lineno + 1))
    total=$((total + 1))
    [[ -z "$line" ]] && continue

    # parse or fail
    echo "$line" | jq -e . >/dev/null 2>&1 || die "line $lineno: malformed JSON" 4

    schema_version="$(echo "$line" | jq -r '.schema_version // "v1"')"

    if [[ "$schema_version" == "v1" ]]; then
      if [[ "$STRICT" == "1" ]]; then
        echo "FAIL line $lineno: --strict and entry is v1 (unsigned)" >&2
        exit 5
      fi
      unsigned=$((unsigned + 1))
      continue
    fi

    if [[ "$schema_version" != "v2" ]]; then
      echo "FAIL line $lineno: unsupported schema_version $schema_version" >&2
      failed=$((failed + 1))
      continue
    fi

    # resolve per-line pubkey if a map is provided
    line_pubkey="$PUBKEY"
    if [[ -n "$PUBKEY_MAP" ]]; then
      signer_id="$(echo "$line" | jq -r '.signer_id')"
      mapped_hex="$(jq -r --arg sid "$signer_id" '.[$sid] // empty' "$PUBKEY_MAP")"
      if [[ -n "$mapped_hex" ]]; then
        line_pubkey="$tmpdir/mapped-$signer_id.pem"
        # raw32 hex → SPKI DER → PEM (Ed25519 SPKI prefix: 302a300506032b6570032100)
        printf '302a300506032b6570032100%s' "$mapped_hex" \
          | xxd -r -p | openssl pkey -inform DER -pubin -out "$line_pubkey" 2>/dev/null \
          || { echo "FAIL line $lineno: mapped pubkey for $signer_id malformed" >&2; failed=$((failed + 1)); continue; }
      fi
    fi

    # build canonical payload: strip envelope fields except signer_pubkey,
    # sort keys at every depth, no whitespace
    canon_file="$tmpdir/canon-$lineno.json"
    echo "$line" | jq -cS '
      del(.signature, .signed_at, .signed_payload_canonical_sha256, .signer_pubkey_fingerprint)
    ' > "$canon_file"

    expected_hash="$(echo "$line" | jq -r '.signed_payload_canonical_sha256')"
    actual_hash="$(openssl dgst -sha256 -binary < "$canon_file" | xxd -p -c 256)"
    if [[ "$actual_hash" != "$expected_hash" ]]; then
      echo "FAIL line $lineno: canonical-hash-mismatch (expected=$expected_hash got=$actual_hash)" >&2
      failed=$((failed + 1))
      continue
    fi

    # ed25519 verify
    sig_hex="$(echo "$line" | jq -r '.signature')"
    sig_bin="$tmpdir/sig-$lineno.bin"
    printf '%s' "$sig_hex" | xxd -r -p > "$sig_bin"

    if openssl pkeyutl -verify -pubin -inkey "$line_pubkey" \
         -rawin -in "$canon_file" -sigfile "$sig_bin" >/dev/null 2>&1; then
      :
    else
      echo "FAIL line $lineno: signature-mismatch" >&2
      failed=$((failed + 1))
      continue
    fi

    # parent-chain check (within-file only — cross-file is a verifier extension)
    self_hash="$actual_hash"
    SEEN_HASHES["$self_hash"]=1
    parent_hash="$(echo "$line" | jq -r '.parent_receipt_sha256 // ""')"
    if [[ -n "$parent_hash" && "$parent_hash" != "null" ]]; then
      if [[ -z "${SEEN_HASHES[$parent_hash]:-}" ]]; then
        echo "FAIL line $lineno: parent-chain-broken (missing parent=$parent_hash)" >&2
        failed=$((failed + 1))
        continue
      fi
    fi

    signed=$((signed + 1))
  done < "$INPUT"

  printf 'total=%d signed=%d unsigned=%d failed=%d\n' "$total" "$signed" "$unsigned" "$failed"
  if (( failed > 0 )); then exit 2; fi
  exit 0
  ```

  ```go
  package signing

  import (
      "bytes"
      "crypto/sha256"
      "encoding/hex"
      "encoding/json"
      "fmt"
      "sort"
  )

  // EnvelopeFields are removed before computing the canonical payload.
  // signer_pubkey IS kept (it is part of the signed body — see spec § 3.2 step 2).
  var EnvelopeFields = []string{
      "signature",
      "signed_at",
      "signed_payload_canonical_sha256",
      "signer_pubkey_fingerprint",
  }

  // Canonicalize takes any JSON-decoded value and produces RFC 8259 bytes with
  // recursively sorted keys, no insignificant whitespace, and UTF-8 encoding.
  // It mirrors `jq -cS` for the shapes used by receipts (no floats, ASCII strings).
  func Canonicalize(v interface{}) ([]byte, error) {
      var buf bytes.Buffer
      if err := encode(&buf, v); err != nil {
          return nil, err
      }
      return buf.Bytes(), nil
  }

  func encode(buf *bytes.Buffer, v interface{}) error {
      switch x := v.(type) {
      case map[string]interface{}:
          keys := make([]string, 0, len(x))
          for k := range x {
              keys = append(keys, k)
          }
          sort.Strings(keys)
          buf.WriteByte('{')
          for i, k := range keys {
              if i > 0 {
                  buf.WriteByte(',')
              }
              kb, err := json.Marshal(k)
              if err != nil {
                  return err
              }
              buf.Write(kb)
              buf.WriteByte(':')
              if err := encode(buf, x[k]); err != nil {
                  return err
              }
          }
          buf.WriteByte('}')
          return nil
      case []interface{}:
          buf.WriteByte('[')
          for i, item := range x {
              if i > 0 {
                  buf.WriteByte(',')
              }
              if err := encode(buf, item); err != nil {
                  return err
              }
          }
          buf.WriteByte(']')
          return nil
      case nil:
          buf.WriteString("null")
          return nil
      case bool, string, json.Number, float64, int, int64:
          b, err := json.Marshal(x)
          if err != nil {
              return err
          }
          buf.Write(b)
          return nil
      default:
          return fmt.Errorf("canonical: unsupported type %T", v)
      }
  }

  // StripEnvelope removes the four envelope fields and returns the receipt
  // body that will be canonicalised and signed.
  func StripEnvelope(receipt map[string]interface{}) map[string]interface{} {
      out := make(map[string]interface{}, len(receipt))
      for k, v := range receipt {
          skip := false
          for _, e := range EnvelopeFields {
              if k == e {
                  skip = true
                  break
              }
          }
          if !skip {
              out[k] = v
          }
      }
      return out
  }

  // CanonicalSHA256 returns the hex-encoded SHA-256 of the canonical bytes
  // (after envelope stripping).
  func CanonicalSHA256(receipt map[string]interface{}) (string, []byte, error) {
      stripped := StripEnvelope(receipt)
      canon, err := Canonicalize(stripped)
      if err != nil {
          return "", nil, err
      }
      sum := sha256.Sum256(canon)
      return hex.EncodeToString(sum[:]), canon, nil
  }
  ```

  ```go
  // LoadKeyBundle reads {dir}/gateway.{key,pub,id}; dir defaults to ~/.config/opsbench/keys.
  func LoadKeyBundle(dir string) (*KeyBundle, error) {
      if dir == "" {
          home, err := os.UserHomeDir()
          if err != nil {
              return nil, err
          }
          dir = filepath.Join(home, ".config", "opsbench", "keys")
      }
      privBytes, err := os.ReadFile(filepath.Join(dir, "gateway.key"))
      if err != nil {
          return nil, fmt.Errorf("read private key: %w", err)
      }
      block, _ := pem.Decode(privBytes)
      if block == nil {
          return nil, errors.New("private key is not PEM-encoded")
      }
      key, err := x509.ParsePKCS8PrivateKey(block.Bytes)
      if err != nil {
          return nil, fmt.Errorf("parse PKCS#8: %w", err)
      }
      priv, ok := key.(ed25519.PrivateKey)
      if !ok {
          return nil, fmt.Errorf("private key is not Ed25519 (got %T)", key)
      }
      pub := priv.Public().(ed25519.PublicKey)

      spki, err := x509.MarshalPKIXPublicKey(pub)
      if err != nil {
          return nil, err
      }
      sum := sha256.Sum256(spki)

      sid, err := os.ReadFile(filepath.Join(dir, "gateway.id"))
      if err != nil {
          return nil, fmt.Errorf("read signer_id: %w", err)
      }

      return &KeyBundle{
          Priv:        priv,
          Pub:         pub,
          SignerID:    strings.TrimSpace(string(sid)),
          Fingerprint: "sha256:" + hex.EncodeToString(sum[:]),
          PubHex:      hex.EncodeToString(pub),
      }, nil
  }

  // Sign mutates the receipt map in place: injects signer_pubkey, computes the
  // canonical payload SHA-256, signs the canonical bytes, and appends the
  // envelope fields. The caller is responsible for ts/gateway_id/agent_class/etc.
  func (kb *KeyBundle) Sign(receipt map[string]interface{}) error {
      receipt["schema_version"] = "v2"
      receipt["signer_id"] = kb.SignerID
      receipt["signer_pubkey"] = kb.PubHex

      hashHex, canon, err := CanonicalSHA256(receipt)
      if err != nil {
          return fmt.Errorf("canonicalise: %w", err)
      }
      sig := ed25519.Sign(kb.Priv, canon)

      receipt["signed_payload_canonical_sha256"] = hashHex
      receipt["signature"] = hex.EncodeToString(sig)
      receipt["signed_at"] = time.Now().UTC().Format(time.RFC3339Nano)
      receipt["signer_pubkey_fingerprint"] = kb.Fingerprint
      return nil
  }

  // Verify checks the canonical SHA-256 and the Ed25519 signature on a receipt
  // already populated with its envelope fields.
  func Verify(receipt map[string]interface{}, pubHex string) error {
      pubBytes, err := hex.DecodeString(pubHex)
      if err != nil || len(pubBytes) != ed25519.PublicKeySize {
          return errors.New("pubkey hex malformed")
      }
      pub := ed25519.PublicKey(pubBytes)

      gotHash, canon, err := CanonicalSHA256(receipt)
      if err != nil {
          return err
      }
      wantHash, _ := receipt["signed_payload_canonical_sha256"].(string)
      if gotHash != wantHash {
          return fmt.Errorf("canonical-hash-mismatch: got=%s want=%s", gotHash, wantHash)
      }
      sigHex, _ := receipt["signature"].(string)
      sig, err := hex.DecodeString(sigHex)
      if err != nil || len(sig) != ed25519.SignatureSize {
          return errors.New("signature hex malformed")
      }
      if !ed25519.Verify(pub, canon, sig) {
          return errors.New("signature-mismatch")
      }
      return nil
  }

  // Compile-time check that PrivateKey satisfies the standard signer interface.
  var _ crypto.Signer = (ed25519.PrivateKey)(nil)
  ```

  ```bash
  # shellcheck shell=bash
  # F3 receipt signing helper, sourced by post-tool-use.sh.
  # Best-effort: any failure here logs to the hook audit and returns 0.
  opsbench_sign_receipt() {
      local incident_dir="$1"
      local agent_class="$2"
      local tool="$3"
      local args_sha="$4"
      local response_sha="${5:-}"
      local decision="${6:-allow}"

      local keys_dir="${HOME}/.config/opsbench/keys"
      local priv="${keys_dir}/gateway.key"
      local sid_file="${keys_dir}/gateway.id"

      command -v openssl >/dev/null 2>&1 || {
          printf '{"ts":"%s","skipped":"openssl-missing"}\n' \
              "$(date -u +%Y-%m-%dT%H:%M:%SZ)" \
              >> /tmp/k8s-incident-post-tool-use.jsonl
          return 0
      }
      [[ -r "$priv" ]] || {
          printf '{"ts":"%s","skipped":"key-missing"}\n' \
              "$(date -u +%Y-%m-%dT%H:%M:%SZ)" \
              >> /tmp/k8s-incident-post-tool-use.jsonl
          return 0
      }

      local sid pub_hex
      sid="$(<"$sid_file")"
      pub_hex="$(openssl pkey -in "$priv" -pubout -outform DER \
                 | tail -c 32 | xxd -p -c 64)"

      local ts canon hash sig_hex fp signed_at
      ts="$(date -u +%Y-%m-%dT%H:%M:%SZ)"
      signed_at="$ts"

      canon="$(jq -cn \
          --arg sv "v2" \
          --arg ts "$ts" \
          --arg gid "" \
          --arg sid "$sid" \
          --arg ac "$agent_class" \
          --arg t  "$tool" \
          --arg a  "$args_sha" \
          --arg r  "$response_sha" \
          --arg dec "$decision" \
          --arg pub "$pub_hex" \
          '{schema_version:$sv, ts:$ts, gateway_id:$gid, signer_id:$sid,
            agent_class:$ac, tool:$t, args_sha256:$a,
            response_sha256: (if $r == "" then null else $r end),
            decision:$dec, signer_pubkey:$pub}' \
          | jq -cS .)"

      local tmpdir
      tmpdir="$(mktemp -d)"
      printf '%s' "$canon" > "$tmpdir/canon.json"
      hash="$(openssl dgst -sha256 -binary < "$tmpdir/canon.json" | xxd -p -c 256)"

      if ! openssl pkeyutl -sign -inkey "$priv" -rawin \
              -in "$tmpdir/canon.json" -out "$tmpdir/sig.bin" 2>/dev/null; then
          rm -rf "$tmpdir"
          printf '{"ts":"%s","skipped":"sign-failed"}\n' "$ts" \
              >> /tmp/k8s-incident-post-tool-use.jsonl
          return 0
      fi
      sig_hex="$(xxd -p -c 256 "$tmpdir/sig.bin")"

      fp="sha256:$(openssl pkey -in "$priv" -pubout -outform DER \
                  | openssl dgst -sha256 -binary | xxd -p -c 256)"

      jq -c --arg h "$hash" --arg s "$sig_hex" \
            --arg fp "$fp" --arg at "$signed_at" \
          '. + {signed_at:$at, signed_payload_canonical_sha256:$h,
                signer_pubkey_fingerprint:$fp, signature:$s}' \
          "$tmpdir/canon.json" \
          >> "$incident_dir/receipts.jsonl"

      rm -rf "$tmpdir"
      return 0
  }
  ```

  ```bash
  #!/usr/bin/env bash
  set -euo pipefail
  KEYS="${HOME}/.config/opsbench/keys"
  [[ -d "$KEYS" ]] || { echo "no keys dir at $KEYS — run install.sh first" >&2; exit 1; }
  [[ ! -f "$KEYS/gateway.key.next" ]] || { echo "gateway.key.next already exists — finish or revert the previous rotation first" >&2; exit 1; }

  openssl genpkey -algorithm Ed25519 -out "$KEYS/gateway.key.next" 2>/dev/null
  chmod 0600 "$KEYS/gateway.key.next"
  openssl pkey -in "$KEYS/gateway.key.next" -pubout -out "$KEYS/gateway.pub.next"
  chmod 0644 "$KEYS/gateway.pub.next"
  if command -v uuidgen >/dev/null 2>&1; then
      uuidgen | tr 'A-Z' 'a-z' > "$KEYS/gateway.id.next"
  else
      python3 -c 'import uuid; print(uuid.uuid4())' > "$KEYS/gateway.id.next"
  fi
  chmod 0644 "$KEYS/gateway.id.next"

  fp_new="$(openssl pkey -in "$KEYS/gateway.key.next" -pubout -outform DER \
            | openssl dgst -sha256 -binary | xxd -p -c 256)"
  fp_old="$(openssl pkey -in "$KEYS/gateway.key" -pubout -outform DER \
            | openssl dgst -sha256 -binary | xxd -p -c 256)"

  cat <<EOF
  Next key staged at $KEYS/gateway.key.next
  Old pubkey fingerprint: sha256:$fp_old
  New pubkey fingerprint: sha256:$fp_new

  Next steps (see docs/key-rotation-runbook.md):
    1. Publish the new fingerprint to auditors out-of-band.
    2. Restart opsbench-gateway gracefully (SIGTERM, wait for in-flight drain).
    3. Re-run any active verifiers with --pubkey-map containing both keys.
    4. Move current → previous, next → current:
         mv $KEYS/gateway.key   $KEYS/gateway.key.prev
         mv $KEYS/gateway.pub   $KEYS/gateway.pub.prev
         mv $KEYS/gateway.id    $KEYS/gateway.id.prev
         mv $KEYS/gateway.key.next $KEYS/gateway.key
         mv $KEYS/gateway.pub.next $KEYS/gateway.pub
         mv $KEYS/gateway.id.next  $KEYS/gateway.id
    5. Restart opsbench-gateway again with the rotated key.
    6. Optionally run \`opsbench-gateway custody verify-signatures --strict\` after T.
  EOF
  ```
