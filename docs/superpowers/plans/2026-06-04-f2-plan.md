# F2 ‚Äî opsbench-gateway (Cedar-evaluated MCP proxy) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax. Each step is sized to 2-5 min of agent work; the largest are the recipe-rewrite loops in Task 11 where the agent fans out across ~30 files. Treat all commands as executed from the repo root unless a step explicitly `cd`s.

**Goal:** Ship `opsbench-gateway` ‚Äî a Cedar-evaluated, custody-log-emitting MCP proxy built by vendoring `stacklok/toolhive` under `packages/opsbench-gateway/` ‚Äî together with the recipe rewrites that point every F0 recipe at the gateway as the recommended (Pi-first) configuration block, keeping the direct-connection block as a documented advanced fallback.

**Architecture:** Single-host, single-tenant Go binary that exposes `127.0.0.1:8765/mcp/<upstream-id>` for each upstream MCP declared in `gateway.yaml`. A request lifecycle is: agent ‚Üí router (vendored toolhive) ‚Üí redactor ‚Üí `cedar-policy/cedar-go` evaluator (loading both F1 policy files, hand-written wins) ‚Üí upstream MCP fan-out ‚Üí custody.log JSONL writer with SHA-256 hashes and redaction preview. Ships as a multi-arch Docker image (`ghcr.io/shaiknoorullah/opsbench-gateway`) and statically-linked binaries on the GitHub Release.

**Tech Stack:** Go 1.22+, Cobra (CLI, inherited from toolhive), `github.com/cedar-policy/cedar-go`, `github.com/spf13/viper` (config), `github.com/fsnotify/fsnotify` (custody rotation only), `github.com/prometheus/client_golang` (`/metrics`), Docker Buildx (multi-arch), cosign (image signing), `goreleaser` (binary release), GitHub Actions, lefthook (existing). Pi integration via `opsbench-gateway pi-snippet` CLI emitter (no Pi MCP server is built ‚Äî Pi calls the gateway over streamable HTTP).

**Parent spec:** [`../specs/2026-06-04-f2-design.md`](../specs/2026-06-04-f2-design.md)
**Parent roadmap:** [`../specs/2026-06-04-f-series-master-roadmap.md`](../specs/2026-06-04-f-series-master-roadmap.md)
**Predecessors:**

- F0 recipe bulk-ship (must be merged ‚Äî F2 rewrites the recipe config blocks F0 created).
- F1 Cedar-for-agents generator (must be merged ‚Äî F2 loads `tools.cedar` + `tools-generated.cedar` from the F1 paths).

- [ ] **1.2 ‚Äî Resolve the upstream pin.** Identify the head commit of `stacklok/toolhive` `main`. Capture both the SHA and date.

  ```bash
  curl -fsSL https://api.github.com/repos/stacklok/toolhive/commits/main \
    | jq -r '.sha + " " + .commit.committer.date' \
    | tee /tmp/toolhive-pin.txt
  ```

  Expected: one line, `<40-char-sha> <ISO-8601 date>`.

- [ ] **1.3 ‚Äî Write the re-runnable vendoring script.** Create `scripts/vendor-toolhive.sh` (mode 0755). It clones at the pinned SHA, copies the source into `packages/opsbench-gateway/internal/upstream/toolhive/`, drops any `.github/`, `.git/`, `docs/`, and `examples/` directories the upstream ships (we don't ingest those), and writes the SHA into `packages/opsbench-gateway/UPSTREAM`.

  ```bash
  #!/usr/bin/env bash
  set -euo pipefail
  REPO_ROOT="$(git rev-parse --show-toplevel)"
  PIN_SHA="${1:?usage: vendor-toolhive.sh <sha>}"
  DEST="$REPO_ROOT/packages/opsbench-gateway/internal/upstream/toolhive"
  WORK="$(mktemp -d)"
  trap 'rm -rf "$WORK"' EXIT
  git clone --filter=blob:none --no-checkout https://github.com/stacklok/toolhive "$WORK/src"
  git -C "$WORK/src" checkout "$PIN_SHA"
  rm -rf "$DEST" && mkdir -p "$DEST"
  rsync -a --exclude='.git' --exclude='.github' --exclude='docs' --exclude='examples' "$WORK/src/" "$DEST/"
  date_iso="$(git -C "$WORK/src" log -1 --format=%cI "$PIN_SHA")"
  printf 'upstream: github.com/stacklok/toolhive\ncommit: %s\ndate: %s\nlast_synced: %s\n' \
    "$PIN_SHA" "$date_iso" "$(date -u +%Y-%m-%dT%H:%M:%SZ)" \
    > "$REPO_ROOT/packages/opsbench-gateway/UPSTREAM"
  ```

  ```go
  package config

  import (
      "fmt"
      "os"
      "regexp"
      "strings"
  )

  var envRE = regexp.MustCompile(`\$\{([A-Z_][A-Z0-9_]*)(:\?)?\}`)

  // Expand replaces ${NAME} / ${NAME:?} tokens with environment values.
  // The :? form errors if NAME is unset.
  func Expand(s string) (string, error) {
      var err error
      out := envRE.ReplaceAllStringFunc(s, func(m string) string {
          subs := envRE.FindStringSubmatch(m)
          name, required := subs[1], subs[2] == ":?"
          v, ok := os.LookupEnv(name)
          if !ok && required {
              err = fmt.Errorf("env var %s is required but unset", name)
              return m
          }
          return v
      })
      if err != nil {
          return "", err
      }
      if strings.Contains(out, "${") {
          return "", fmt.Errorf("unresolved env token in %q", s)
      }
      return out, nil
  }
  ```

- [ ] **3.3 ‚Äî Write `internal/custody/canonical.go`.** Function `Canonicalize(v any) ([]byte, error)` returns RFC 8785-style canonical JSON (keys sorted, no insignificant whitespace, numbers in shortest form). This is what feeds the SHA-256 hash. Implement using `encoding/json` with `json.Marshal` after recursively sorting maps.

  ```go
  package custody

  import (
      "bytes"
      "encoding/json"
      "sort"
  )

  func Canonicalize(v any) ([]byte, error) {
      switch x := v.(type) {
      case map[string]any:
          keys := make([]string, 0, len(x))
          for k := range x {
              keys = append(keys, k)
          }
          sort.Strings(keys)
          var buf bytes.Buffer
          buf.WriteByte('{')
          for i, k := range keys {
              if i > 0 {
                  buf.WriteByte(',')
              }
              kb, _ := json.Marshal(k)
              buf.Write(kb)
              buf.WriteByte(':')
              vb, err := Canonicalize(x[k])
              if err != nil {
                  return nil, err
              }
              buf.Write(vb)
          }
          buf.WriteByte('}')
          return buf.Bytes(), nil
      case []any:
          var buf bytes.Buffer
          buf.WriteByte('[')
          for i, e := range x {
              if i > 0 {
                  buf.WriteByte(',')
              }
              eb, err := Canonicalize(e)
              if err != nil {
                  return nil, err
              }
              buf.Write(eb)
          }
          buf.WriteByte(']')
          return buf.Bytes(), nil
      default:
          return json.Marshal(x)
      }
  }
  ```

  This is the largest file in PR 4. Keep it under 600 lines by delegating envelope parsing to toolhive's own helpers (call through `internal/upstream/toolhive` exports ‚Äî that's the patch the next step adds).
